Установка Office 2016 Pro Plus

Полный цикл развертывания через ODT.

ШАГ 0Вход в папку
POWERSHELL
Set-Location -Path "C:\ODT"
Выполнить
ШАГ 1-5Установка
POWERSHELL
# 1. Создание папки New-Item -Path "C:\ODT" -ItemType Directory -Force; Set-Location "C:\ODT" # 2. Загрузка ODT Invoke-WebRequest "https://go.microsoft.com/fwlink/p/?LinkID=626065" -OutFile "odt.exe"; .\odt.exe /quiet /extract:C:\ODT # 3. Конфигурация Set-Content "configuration.xml" '<Configuration><Add OfficeClientEdition="64" Channel="Current"><Product ID="ProPlusRetail"><Language ID="ru-ru"/></Product></Add><Display Level="None" AcceptEULA="TRUE"/></Configuration>' # 4. Установка .\setup.exe /configure configuration.xml
Выполнить
Ожидаемый результат: Office 2016 установлен.
ШАГ 6-8Активация
POWERSHELL
cscript "C:\Program Files\Microsoft Office\Office16\OSPP.VBS" /inpkey:XQNVK-8JYDB-WJ9W3-YJ8YR-WFG99 cscript "C:\Program Files\Microsoft Office\Office16\OSPP.VBS" /sethst:kms.digiboy.ir cscript "C:\Program Files\Microsoft Office\Office16\OSPP.VBS" /act
Выполнить
Ожидаемый результат: <Product activation successful>
ШАГ 10Сброс активации
POWERSHELL
cscript "C:\Program Files\Microsoft Office\Office16\OSPP.VBS" /dstatus cscript "C:\Program Files\Microsoft Office\Office16\OSPP.VBS" /unpkey:XXXXX
Выполнить
ШАГ 11Полная очистка (Hard Uninstall)
POWERSHELL
# 1. Удаление Appx Get-AppxPackage -Name *Microsoft.Office.Desktop* | Remove-AppxPackage -ErrorAction SilentlyContinue # 2. Очистка реестра Remove-Item -Path "HKLM:\SOFTWARE\Microsoft\Office\ClickToRun" -Recurse -Force Remove-Item -Path "HKLM:\SOFTWARE\Microsoft\AppVISV" -Recurse -Force Remove-Item -Path "HKCU:\Software\Microsoft\Office" -Recurse -Force # 3. Проверка ключей Get-CimInstance SoftwareLicensingProduct -Filter "PartialProductKey IS NOT NULL" | Where-Object { $_.Name -like "*Office*" }
Выполнить
Ожидаемый результат: Система очищена от Office.

Установка Visio (2016 / 2021)

Установка дополнительного компонента Visio Professional.

ШАГ 1Установка и Активация
POWERSHELL
# Visio 2016 Set-Location "C:\ODT"; Set-Content "visio16.xml" '<Configuration><Add OfficeClientEdition="64"><Product ID="VisioProRetail"><Language ID="ru-ru"/></Product></Add><Display Level="Full" AcceptEULA="TRUE"/></Configuration>'; .\setup.exe /configure visio16.xml cscript "C:\Program Files\Microsoft Office\Office16\OSPP.VBS" /inpkey:PD3PC-RHNGV-FXJ29-8JK7D-RJRJK cscript "C:\Program Files\Microsoft Office\Office16\OSPP.VBS" /act
Выполнить
Ожидаемый результат: Visio 2016 установлен и активирован.
ШАГ 3Удаление Visio
POWERSHELL
# Удаление Visio Set-Location "C:\ODT"; Set-Content "remove_visio.xml" '<Configuration><Remove><Product ID="VisioProRetail"/><Product ID="VisioPro2021Retail"/></Remove><Display Level="None" AcceptEULA="TRUE"/></Configuration>'; .\setup.exe /configure remove_visio.xml
Выполнить
Ожидаемый результат: Visio удален.

УДАЛЕНИЕЖесткое удаление (Ярлыки + Файлы)
POWERSHELL
# 1. Закрытие процессов Stop-Process -Name "OUTLOOK", "MSACCESS", "ONENOTE" -Force -ErrorAction SilentlyContinue # 2. Удаление ярлыков (ProgramData) Remove-Item "C:\ProgramData\Microsoft\Windows\Start Menu\Programs\Access.lnk" -Force -ErrorAction SilentlyContinue Remove-Item "C:\ProgramData\Microsoft\Windows\Start Menu\Programs\Outlook.lnk" -Force -ErrorAction SilentlyContinue Remove-Item "C:\ProgramData\Microsoft\Windows\Start Menu\Programs\OneNote.lnk" -Force -ErrorAction SilentlyContinue # 3. Удаление компонентов из Office (Перенастройка) Set-Location "C:\ODT" Set-Content "exclude_apps.xml" '<Configuration><Add OfficeClientEdition="64" Channel="Current"><Product ID="ProPlus2021Retail"><Language ID="ru-ru"/><ExcludeApp ID="Outlook"/><ExcludeApp ID="Access"/><ExcludeApp ID="OneNote"/></Product></Add><Display Level="None" AcceptEULA="TRUE"/></Configuration>' .\setup.exe /configure exclude_apps.xml
Выполнить
Ожидаемый результат: Ярлыки исчезнут, приложения удалятся.
